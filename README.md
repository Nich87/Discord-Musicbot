# Discord-Musicbot

**mainブランチはdiscord.js(v12.5.3)向けです。**  
**v12のコードはサポートを打ち切っています。**  
**v13のバグはIssueもしくはPRを頂ければ嬉しいです。**  

## 動作環境
> Node.js : 16.13.0  
> OS:x64 Ubuntu 20.04  x64 Windows11  

## 必須モジュール(ライブラリ) 
- @discordjs/voice
- @discordjs/opus
- ffmpeg-static
- ytdl-core/yt-search
- tweetnacl
